Persistence Package

This package contains persistence layer related classes for the Portal project.

Sub-packages:

  • orm - Object-Relational Mapping functionality
  • config - Persistence configuration classes
  • example - Usage examples

Design Principles:

  • Facade pattern to hide third-party library details
  • Module-based package naming (not library-based)
  • Unified persistence API
  • Configuration-driven approach
  • Reflection-based implementation to avoid direct dependencies

Key Features:

  • Entity management operations (save, update, delete, find)
  • Transaction management
  • Query building
  • Native SQL execution
  • Multi-data source support
  • Configuration management
