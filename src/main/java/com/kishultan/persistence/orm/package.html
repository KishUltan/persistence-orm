ORM Package

This package contains Object-Relational Mapping functionality using facade pattern.

Classes:

  • EntityManager - Facade for entity management operations
  • PersistenceManager - Manager for EntityManager lifecycle
  • InternalOrmFactory - Internal factory for ORM server creation (package-private)

Usage:

// Get default entity manager
EntityManager em = PersistenceManager.getDefaultManager();

// Get entity manager with specific data source
EntityManager em = PersistenceManager.getManager("myDataSource");

// Save entity
User user = new User();
user.setName("John");
em.save(user);

// Find entity
User found = em.findById(User.class, 1L);

// Create query
Object query = em.createQuery(User.class);
// Use reflection to build query...

Design Benefits:

  • Hides third-party library implementation details
  • Provides unified API for persistence operations
  • Supports multiple data sources
  • Configuration-driven approach
  • Reflection-based implementation avoids compile-time dependencies
