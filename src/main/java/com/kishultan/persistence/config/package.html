Persistence Config Package

This package contains configuration classes for the persistence layer.

Classes:

  • PersistenceConfig - Configuration for persistence settings

Usage:

// Get default configuration
PersistenceConfig config = PersistenceConfig.getDefaultConfig();

// Get development configuration
PersistenceConfig devConfig = PersistenceConfig.getDevelopmentConfig();

// Get production configuration
PersistenceConfig prodConfig = PersistenceConfig.getProductionConfig();

// Custom configuration
PersistenceConfig customConfig = new PersistenceConfig();
customConfig.setPackageScan("com.mycompany");
customConfig.setSlowQueryLogging(true);
customConfig.setSlowQueryThreshold(1000);

Configuration Options:

  • DDL generation and execution
  • Package scanning
  • Query plan logging
  • Slow query logging
  • Performance thresholds
